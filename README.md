# Telegram-бот для командных задач

Простой Telegram-бот на Python для общего списка задач команды. Участники добавляют задачи, просматривают список в чате или скачивают его в формате CSV.

**Стек:** [aiogram 3](https://docs.aiogram.dev/), SQLite3, python-dotenv.

## Возможности

| Команда | Описание |
|---------|----------|
| `/start` | Приветствие и справка по командам |
| `/add <текст>` | Добавить задачу в общий список |
| `/list` | Показать все задачи в чате |
| `/list_csv` | Отправить файл `tasks.csv` со всеми задачами |

## Структура проекта

```
.
├── main.py                 # Точка входа, запуск polling
├── config.py               # Загрузка BOT_TOKEN из .env
├── test_db.py              # Проверка базы данных без бота
├── requirements.txt
├── .env                    # Токен бота (не коммитить)
├── .gitignore
├── tasks.db                # База SQLite (создаётся при запуске)
├── database/
│   └── db.py               # Таблица tasks, добавление и чтение
├── handlers/
│   ├── __init__.py         # Регистрация роутеров
│   └── commands.py         # Обработчики команд
└── keyboards/              # Заготовка под кнопки (в примере не используется)
```

## База данных

Таблица `tasks`:

| Поле | Тип | Описание |
|------|-----|----------|
| `id` | INTEGER | Первичный ключ |
| `text` | TEXT | Текст задачи |
| `user` | TEXT | Автор (`@username` или Telegram ID) |
| `created_at` | TEXT | Дата и время создания |

Файл базы: `tasks.db` в корне проекта.

## Требования

- Python 3.10+
- Токен бота от [@BotFather](https://t.me/BotFather)

## Установка и запуск

### 1. Клонирование / переход в папку проекта

```bash
cd "/Users/pavelkoff/Desktop/Урок_VPb06"
```

### 2. Виртуальное окружение

```bash
python3 -m venv .venv
source .venv/bin/activate   # macOS / Linux
# .venv\Scripts\activate    # Windows
```

### 3. Зависимости

```bash
pip install -r requirements.txt
```

### 4. Настройка токена

```bash
cp .env.example .env
```

Откройте `.env` и укажите токен:

```
BOT_TOKEN=ваш_токен_от_BotFather
```

> Файл `.env` не коммитьте в git — он уже в `.gitignore`.

### 5. Проверка базы данных (опционально)

```bash
python test_db.py
```

Скрипт создаст таблицу, добавит тестовую задачу и выведет список записей.

### 6. Запуск бота

```bash
python main.py
```

В консоли появится сообщение `Бот запущен`. Остановка: `Ctrl+C`.

### 7. Проверка в Telegram

Найдите бота по имени, которое задали в BotFather, и отправьте:

```
/start
/add Купить кофе для офиса
/list
/list_csv
```

## Примеры использования

**Добавить задачу:**

```
/add Подготовить отчёт к пятнице
```

**Пустая команда `/add`** — бот попросит указать текст:

```

  Пример: /add Подготовить отчёт к пятнице
```

## Устранение неполадок

| Проблема | Решение |
|----------|---------|
| `Не задан BOT_TOKEN` | Создайте `.env` из `.env.example` и вставьте токен |
| Бот не отвечает | Проверьте, что `python main.py` запущен без ошибок |
| Команды не видны в меню | В BotFather можно задать список команд вручную (необязательно) |
